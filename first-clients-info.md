## 자주 묻는 질문 (FAQ, 5개)

```
Q1. 도메인이나 호스팅이 없는데 가능한가요?
A1. 네, 가능합니다. 도메인 구매 및 호스팅 세팅을 안내해 드립니다. 도메인은 연 1~2만원, 호스팅은 월 500~5,000원 수준입니다. 무료 호스팅(GitHub Pages, Vercel 등)도 가능합니다.

Q2. 제작 후 내용을 직접 수정할 수 있나요?
A2. 네, 소스 코드를 전부 전달해 드리므로 직접 수정하실 수 있습니다. 간단한 수정 가이드 문서도 함께 제공합니다. 코딩 지식이 없으셔도 텍스트나 이미지 교체는 안내 따라 가능합니다.

Q3. DB수집 폼은 어떻게 작동하나요?
A3. 방문자가 폼에 정보를 입력하면, 지정하신 이메일 또는 구글 스프레드시트로 자동 전송됩니다. 별도 서버 비용 없이 무료로 운영 가능합니다.

Q4. 아임웹이나 워드프레스로 제작하나요?
A4. 아닙니다. HTML/CSS/JavaScript로 직접 코딩하여 제작합니다. 플랫폼 종속 없이 어디서든 자유롭게 운영 가능하며, 월 구독료가 없습니다.

Q5. 작업 기간은 얼마나 걸리나요?
A5. STANDARD 패키지 기준 3일, DELUXE 5일, PREMIUM 7일입니다. 자료가 미리 준비되어 있으면 더 빠른 작업도 가능합니다.
```
